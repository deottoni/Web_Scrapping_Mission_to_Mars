# Web Scrapping: Mission to Mars

## Description:
This project consisted in scrapping data related to Mars from a variety of sources (i.e. twitter, nasa etc), then create a single html page to display all that data.


## Technologies used:
* Python
* Pandas
* BeautifulSoup
* Splinter
* Flask
* PyMongo
* MongoDB
* html/CSS
* Bootstrap
